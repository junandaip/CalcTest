# Calculator App
Simple Calculator to calculate your needs

## Environment Setup
[Java JDK 8](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html).

[Gradle 6.7](https://gradle.org/install/)

## How To Build
    gradlew build
    
## How To Run
    gradlew run
